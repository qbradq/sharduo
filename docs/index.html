Trammel Time

Welcome to Trammel Time, an experimental Ultima Online server emulation project. The project is currently in an open alpha status. There is no guarantee that a server wipe will not take place before the start of the beta phase if the project gets that far.

Feedback is especially appreciated. Please join the Discord server to join the community, report bugs and request future features.

Connecting

Please download the client files and extract them to a convient location on your computer. On Windows computers, double-click the shortcut included with the client to start the game and enter Britannia.

On other operating systems - or as an alternative on Windows - please download and run the ClassicUO launcher. Create a profile and configure it to connect to a server at IP address 15.204.238.7 and port 7775. Please do not use any plugins. They are technically allowed but not officially supported.

Game Server Rules

  • Please note that the service is in a testing phase. Many of these rules will completely reverse if a production release happens. Such rules are in italics.
  • All of the rules of the Discord community apply to all forms of public communication within the game.
  • Do not exploit bugs in the game service. If found please report the bug in the ⁠bug-reports channel.
  • Third-party programs such as Razor Enhanced are allowed as are any other clients. However only the ClassicUO client without plugins is officially supported. Please if you think to yourself, "gee this is tedious, I'd like a macro for it" report it as a bug in the ⁠bug-reports channel. Tedium is a design failure.
  • No limit on accounts per person.
  • No limit on accounts per household. Please note that suspensions and bans are IP-wide.
  • Multi-boxing is allowed.
  • Unattended resource gathering is allowed.
  • Never share your account with another person. Actions taken on your account can result in administrative action and we do not accept "it was someone else playing my character" as an excuse.
  • Staff may create items, alter decor, place permanent objects, etc. in response to player requests.

Direction of the Shard

  • This shard will never allow nonconsensual PvP.
  • Consensual PvP through guild wars and PvP events are planned but are not the focus of development.
  • Development will be focused initially on crafting followed by PvE mechanics and content.
  • Game mechanics are being written from the ground up. The goal is to implement each feature in such a way that feels like the UO:R-era mechanics integrated with quality of life features added from latter eras or games.
  • Trammel Time is implemented using open-source software available at https://github.com/qbradq/sharduo. Contributions and forks are welcome but please understand the implications of the AGPLv3 license. If you use the software to offer another service you will have to share all of your source code, even if you do not release a binary. This includes decorations, region definitions, spawns, signs, doors, new and unique items, monsters, equipment and all manor of creative content.
  • Trammel Time will never be operated as a for-profit business.
